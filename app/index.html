Node.js installatie

Node.js is een javascript-implementatie om javascript zonder browser te kunnen draaien.

Om de Node.js interpreter werkend te krijgen zijn er bepaalde stappen nodig die uitgevoerd moet worden.

Ga naar de opgegeven link klik hier en selecteer de versie open source.

Git status

Met deze commando zie je de voortgang van je afgelopen acties. Met name de bestanden die zijn gewijzigd.

Na dat u deze commando heeft ingevoerd kunt u aan de hand van de informatie zien dat de index.html en style.css zijn gewijzigd.

jeremiah@office19 ~/beginnen-met-git>$ git status
# On branch master
# Changes not staged for commit:
# (use "git add ..." to update what will be committed)
# (use "git checkout -- ..." to discard changes in working directory)
#
# modified: app/index.html
# modified: app/styles/main.css

Git add

Met git add voeg je de veranderingen toe aan de locale host.

Na het toevoegen van de veranderingen check met de volgende commando of het is gewerkt. $git status

jeremiah@office19 ~/beginnen-met-git> git add app/index.html
jeremiah@office19 ~/beginnen-met-git> git status
# On branch master
# Changes to be committed:
# (use "git reset HEAD ..." to unstage)
# modified: app/index.html
# Changes not staged for commit:
# (use "git add ..." to update what will be committed)
# (use "git checkout -- ..." to discard changes in working directory)
# modified: app/styles/main.css

node.js

Ophalen

Als eerste moeten we het archief met de source ophalen, met de volgende link klik hier

Uitpakken

In de terminal maak je een map tools aan met

$mkdir tools vervolgt door $cd tools en tot slot $tar xvzf ~/Downloads/node-v0.10.26.tar.gz

Configureren

Navigeer de juist uitgepakte map in. $cd ~/tools/node-v0.10.26/Door de volgende commando worden de bestanden in de juiste map geinstalleerd. $./configure --prefix=~/tools

Bouwen

$make

Installeren

$make install

Uitproberen

Je hebt nu een javascript implementatie buiten de browser

$node

Verlaten met Ctrl + d
